# CS417-Minimal-Sim

## Description
This project implements a Korean bar–themed incremental idle simulation set in a stylized bar environment. Players generate resources represented by beer bottles and beer boxes, which increase over time through Euler-integrated ramping resources. Through XR interactions, players can deploy generators that increase the resource rate but cost resources, with exponentially increasing costs, and purchase power-ups that multiply generator output. Players can also manually increment resources, unlock the second resource UI, earn achievement trophies, and use a prestige system to reset progress for permanent bonuses. The simulation also supports inter-session saves and idle progress, allowing resources to accumulate even when the game is closed.
